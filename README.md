# Blockchain for Secure Transactions

This is a decentralized blockchain application developed using Python and Flask. It supports:

- Secure user registration and login
- Wallet-based transactions
- Mining using Proof-of-Work (PoW)
- Blockchain validation and network sync
- File-based JSON ledger storage
- Peer-to-peer multi-node functionality


## Tech Stack

- Python, Flask
- HTML, CSS (Frontend)
- JSON for file-based storage
- Peer node architecture


## To Run

```bash
pip install -r requirements.txt
python app.py
